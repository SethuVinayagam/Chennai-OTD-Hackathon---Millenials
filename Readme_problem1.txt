Assumptions 
(1) Only hours and minutes is taken as input from user for time. Seconds is taken as 00.
Example
Input - 14:20
We consider it as 14:20:00
(2) Stops entered by user is case sensitive need to given exact names.
(3) There are few stops names having multiple ids. Only 6 outputs for each stop id will be displayed.
Example: Chemmancheri has 2 stop IDs so total of 12 buses will be displayed in the output.
